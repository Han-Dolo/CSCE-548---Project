CSCE 548 Project 4 Deployment and System Test

Project Summary

This repository contains a four-layer campus event RSVP application with a MySQL data layer, Java business and service layers, a Spring Boot REST API, and a browser frontend hosted from the same application.

Prerequisites

  • Java 17
  • Maven 3.9+
  • MySQL Server 8+
  • An IDE or terminal environment

Download and Open

  1. Download the repository ZIP from GitHub or clone it.
  2. Extract the archive.
  3. Open the project root that contains pom.xml, sql/, and src/.

Database Setup

mysql -u root -p < sql/schema.sql
mysql -u root -p < sql/seed.sql

Confirm setup:

SHOW DATABASES;
USE campus_rsvp;
SHOW TABLES;
SELECT COUNT(*) FROM venues;
SELECT COUNT(*) FROM attendees;
SELECT COUNT(*) FROM events;
SELECT COUNT(*) FROM rsvps;

Environment Configuration

The application reads these environment variables:

  • DB_URL default: jdbc:mysql://localhost:3306/campus_rsvp
  • DB_USER default: root
  • DB_PASSWORD default: empty
export DB_URL=jdbc:mysql://localhost:3306/campus_rsvp
export DB_USER=root
export DB_PASSWORD=your_password

Build and Host

mvn -q -DskipTests package
mvn -q spring-boot:run -Dspring-boot.run.mainClass="com.campus.rsvp.api.RsvpApiApplication"

After startup:

  • API base URL: http://localhost:8080
  • Frontend URL: http://localhost:8080/
  • API check: http://localhost:8080/api/events
